# Code of Conduct

## Our Commitment

We want the TaktX Community Console project to be a welcoming, respectful, and constructive open-source community for everyone.

Participants in this project are expected to:

- Be respectful and considerate in language and behavior
- Assume good intent and discuss disagreements constructively
- Focus on what is best for the project and community
- Welcome newcomers and help make contribution paths clear
- Give and receive feedback professionally

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, intimidation, threats, or personal attacks
- Discriminatory or demeaning language or conduct
- Trolling, repeated disruption, or bad-faith engagement
- Publishing private information without permission
- Any other conduct that a reasonable person would consider inappropriate in a professional open-source community

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, code reviews, and other community interactions related to this repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

Maintainers may also temporarily or permanently restrict participation for repeated or severe violations.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the project maintainers through a non-public channel available on the repository hosting platform.

If a private reporting mechanism is not available, do not post sensitive details publicly. Instead, open a minimal issue requesting a secure contact channel.
